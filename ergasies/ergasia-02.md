# 2η Εργασία

Η δεύτερη εργασία αφορά στη δηµιουργία και διαχείριση κλάσεων και αντικειµένων. Η εργασία επίσης εξετάζει τη δυνατότητά σας στην αντικειµενοστρεφή σχεδίαση προγραµµάτων. Το αντικείµενο της 2ης εργασίας είναι η διαχείριση δεξαµενών καυσίµων.

Για κάθε δεξαµενή καταχωρούµε τις παρακάτω πληροφορίες:

- **Είδος καυσίµου** (π.χ. βενζίνη, πετρέλαιο, κ.λπ)
- **Τιµή καυσίµου** (π.χ. 1.75€)
- **Συνολική χωρητικότητα** της δεξαµενής σε λίτρα (π.χ. 1000)
- **Ποσότητα καυσίµου** που υπάρχει στη δεξαµενή σε λίτρα (π.χ 720)

Το **Είδος καυσίµου** προσδιορίζει το είδος του καυσίµου που περιέχει η δεξαµενή και µπορεί να περιέχει γράµµατα και αριθµούς.

Η **Τιµή καυσίµου** προσδιορίζει τη τιµή του καυσίµου της δεξαµενής για ένα λίτρο.

Η **Συνολική χωρητικότητα** περιέχει τον αριθµό σε λίτρα της **χωρητικότητας** της δεξαµενής.

Η **Ποσότητα καυσίµου** περιέχει τον αριθµό σε λίτρα της ποσότητας του καυσίµου που **υπάρχει** στη δεξαµενή.

## 1o βήμα

Καταστρώστε µία κλάση η οποία να προσοµοιάζει **αντικείµενα δεξαµενές**. Ονοµάστε τη κλάση **dexameni**. Η κλάση πρέπει να διαθέτει τις κατάλληλες µεταβλητές-µέλη για την αποθήκευση:

- **Είδος καυσίµου**
- **Τιµή καυσίµου**
- **Συνολική χωρητικότητα**
- **Ποσότητα καυσίµου**

Αποφασίστε εσείς για τους τύπους και τα ονόµατα των µεταβλητών-µέλη που θα χρησιµοποιήστε. Φροντίστε τα ονόµατα να υποδηλώνουν τη πληροφορία που θα καταχωριστεί και να είναι του κατάλληλου τύπου.

## 2o βήμα

Η κλάση **dexameni** θα πρέπει να διαθέτει µεθόδους µε τα ακόλουθα ονόµατα και λειτουργίες:

- Μέθοδο **δόμησης**
  - Η µέθοδος δόµησης δεν θα έχει παραµέτρους και θα δίνει αρχικές τιµές µηδέν (0) στη **Ποσότητα καυσίµου**, µηδέν (0) στη **Τιµή καυσίµου** και χίλια (1000) στη **Συνολική χωρητικότητα**.
- Μέθοδο **axia()**
  - Η µέθοδος axia() **δεν** θα έχει παράµετρο και θα **επιστρέφει** ως τιµή τησυνολική αξία του καυσίµου της δεξαµενής. Η αξία υπολογίζεται αν **πολλαπλασιάσουµε** τη **Τιµή καυσίµου** µε τη **Ποσότητα καυσίµου** της δεξαµενής. Η µέθοδος **δεν** θα εµφανίζει τίποτα στην οθόνη.
- Μέθοδο **info()**
  - Η µέθοδος αυτή δεν θα έχει παράµετρο και θα εµφανίζει στην οθόνη την συνολική κατάσταση της δεξαµενής στην ακόλουθη µορφή:

    ```c++
    Είδος καυσίµου
    MAX: Συνολική χωρητικότητα
    ===============================
    Timi : Τιµή καυσίµου
    Posotita : Ποσότητα καυσίµου
    ===============================
    Axia : Αξία καυσίµου
    *******************************
    ```

    ***ΠΡΟΣΟΧΗ**: Ότι είναι γραµµένο µε **πλάγια γραφή** προέρχεται από τις µεταβλητές-µέλη ή από τις µεθόδους (συναρτήσεις-µέλη) του αντικειµένου.*

## 4o βήμα

Τώρα θα πρέπει στη κλάση **dexameni** να προσθέσουµε και άλλες µεθόδους µε τα ακόλουθα ονόµατα και λειτουργίες:

- Μέθοδος **prosthiki()**
  - Η µέθοδος αυτή θα δέχεται ως παράµετρο µια ποσότητα (σε λίτρα) και θα τα **προσθέτει** στη δεξαµενή (θα προστίθενται στη **Ποσότητα καυσίµου**). Η µέθοδος **δεν** θα ζητάει τίποτα από τον χρήστη ! Η µέθοδος πρέπει **πριν** να προσθέσει τη ποσότητα στη δεξαµενή, να **ελέγχει** αν αυτή τη ποσότητα **χωράει** στον υπόλοιπο χώρο της δεξαµενής και αν όχι να εµφανίζει το µήνυµα `"Den ypaxrxei arketos xoros gia ## litra"` και να **µην** προσθέτει τη ποσότητα. Στη θέση του `##` θα πρέπει να εµφανίζονται τα λίτρα που προσπαθήσαµε να προσθέσουµε.
- Μέθοδος **aferesi()**
  - Η µέθοδος αυτή θα δέχεται ως παράµετρο µια ποσότητα (σε λίτρα) και θα τα **αφαιρεί** από τη δεξαµενή (θα αφαιρούνται από τη **Ποσότητα καυσίµου**). Η µέθοδος **δεν** θα ζητάει τίποτα από τον χρήστη ! Η µέθοδος πρέπει **πριν** να αφαιρέσει τη ποσότητα από τη δεξαµενή να **ελέγχει** αν αυτή η ποσότητα καυσίµου υπάρχει στη δεξαµενή και αν όχι να εµφανίζει το µήνυµα `"Den ypaxrxoyn ## litra $$ sti dexameni"` και να **µην** αφαιρεί τη ποσότητα. Στη θέση του `##` θα πρέπει να εµφανίζονται τα λίτρα που προσπαθήσαµε να αφαιρέσουµε και στη θέση του `$$` το είδος καυσίµου που υπάρχει στη δεξαµενή

Αυτή η εργασία µπρεί να γίνει ανά ζεύγη φοιτητών, όπως και η προηγούµενη. Αν όµως κάποιος δεν µπορεί, ή δεν θέλει να βρεί κάποιον συµφοιτητή του, µπορεί να την κάνει ατοµικά.

Σκοπός της οµαδικής εργασίας είναι η µεταφορά γνώσης από κάποιους συναδέλφους σας στους υπόλοιπους.

Αν δεν µπορέσετε να ολοκληρώστε το πρόγραµµα, στείλτε το µέχρι το σηµείο που έχετε φτάσει αρκεί να µεταγλωττίζεται σωστά.
